z-index usage:
0: background layer 1
1: background layer 2
2: bacground special effects
3: occlusion effects 1 (shadow, background-foreground merge, etc.)
4: main contents layer 1 (other)
5: main contents layer 2 (video/text/images)
6: main contents layer 3 (interactive)
7: occlusion effects 2 (like cutouts)
8: menu elements layer 1
9: menu elements layer 2
10: special effects
11: popup occlusion
12: popups
